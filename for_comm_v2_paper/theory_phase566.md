# Phase 566: Information theory — MIMO + polar + LDPC

**MIMO (Foschini 1996 + Telatar 1999)**: N×N MIMO capacity scales linearly with min(N_T, N_R)。**LDPC codes (Gallager 1963, rediscovered MacKay 1996)**: Shannon-limit-approaching、5G NR data channels。**Turbo codes (Berrou 1993)**: 0.5 dB from Shannon limit、3G/4G LTE。**Polar codes (Arikan 2009)**: provably capacity-achieving、5G NR control channels。**LDPC + Polar 5G consensus**。**6G candidates**: Plotkin construction, AI-designed codes (DeepMind 2023)。
