```
ρ_energy = density over (sources × storage × conversion × consumers × grid)

```

### H_EN2: K_energy diversity = grid resilience

```
Monoculture grid (e.g., 90% coal): K_energy low → fragile
Diverse grid (renewables + storage + nuclear): K_energy high → resilient

```

```

ITU view: learning curve = K_information about manufacturing
```
