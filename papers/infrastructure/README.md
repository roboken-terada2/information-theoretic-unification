$$K_{\text{infrastructure}} = K_{\text{capital}} \otimes K_{\text{network}} \otimes K_{\text{maintenance}}, \quad \frac{dK_{\text{cap}}}{dt} = K_{\text{invest}} - K_{\text{decay}}$$
