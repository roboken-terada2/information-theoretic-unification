# Phase 445: K_psych = -log ρ_mental_state 厳密定義

**ρ_mental tensor product**: ρ_neural ⊗ ρ_cognitive ⊗ ρ_affective ⊗ ρ_behavioral。**Neural**: connectome dynamics、Default Mode Network (DMN, Raichle 2001)。**Cognitive**: working memory、attention、executive function。**Affective**: PANAS scale、circumplex model (Russell 1980 valence-arousal)。**Behavioral**: phenotyping (passive sensing, EMA)。**K_psych = -log ρ_mental** modular Hamiltonian。⟨K_psych⟩ = von Neumann entropy of mental state。**Measurement**: fMRI default mode dynamics、HRV、EEG、smartphone passive sensing (Mindstrong)。
