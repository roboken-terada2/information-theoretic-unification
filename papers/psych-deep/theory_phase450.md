# Phase 450: Autism spectrum + ADHD

**Autism Spectrum Disorder (ASD, DSM-5 2013 unified)**: ~75M global、social communication + restricted/repetitive behaviors。**Prevalence increase**: 1 in 36 children US 2023 (CDC) (up from 1 in 150 in 2000)、better diagnosis + true increase debated。**Genetic**: 800+ genes implicated (SFARI)、heritability ~80%。**ADHD**: ~366M (incl adults)、stimulants (methylphenidate, amphetamine)、non-stimulants (atomoxetine, viloxazine 2021)。**ITU view**: ASD = ρ_mental の selective sensory/cognitive subspace specialization、K_psych structure ≠ pathological but neurotypical-different。
