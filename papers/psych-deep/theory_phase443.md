## Tier 1+ #7 の 4 つの中心仮説 (H_P1-H_P4)

### H_P1: Mental state is operator-algebraic state

```
個人 P の mental state は density operator ρ_mental で記述:
  ρ_mental = density over neural-cognitive-emotional configurations
  H_mental = L²(Neural × Cognitive × Affective × Behavioral)
```

### H_P2: Pathology ⇔ ρ_mental の subspace 局在

```
Depression (MDD): ρ_mental が negative valence subspace に局在
  ⟨K_psych⟩_MDD < ⟨K_psych⟩_healthy
  (rigidity, anhedonia, narrow affect range)

Anxiety/PTSD: ρ_mental が threat-detection subspace に hyperactive
Schizophrenia: ρ_mental が delusion subspace に "stuck"
ASD: ρ_mental の selective subspace specialization
```

### H_P3: Therapy = ρ_mental modular flow re-distribution

```
Antidepressant (SSRI): slow ρ_mental shift (weeks)
Ketamine: rapid ρ_mental phase transition (hours)
Psychedelic: ρ_mental decoherence + reorganization
TMS/DBS: targeted ρ_mental subspace activation
Psychotherapy: ρ_mental long-term Bayesian update

ITU view: 各治療は modular flow σ_t を異なる方向に適用
```

### H_P4: Psychedelics = K_psych entropy maximization

```
**Carhart-Harris "Entropic Brain" hypothesis (2014 Front Hum Neurosci)**:
  Psychedelic state = high-entropy brain state
  ⟨K_psych⟩^{psychedelic} >> ⟨K_psych⟩^{normal}

ITU operator-algebraic version:
  Psilocybin/LSD/DMT applied modular flow disrupts ρ_mental
  → ρ_mental_high_entropy → re-equilibration to healthier state

Therapeutic mechanism:
  Pathological ρ_mental → high-K ψ-state → healthier ρ_mental
  Like "annealing" in statistical physics
```
