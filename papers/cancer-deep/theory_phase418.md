# Phase 418: Metastasis modular flow

**Metastatic cascade**: invasion → intravasation → circulation → extravasation → colonization。**Stephen Paget 1889 'seed and soil'** theory、Joan Massagué Memorial Sloan Kettering modern lab (Nat Rev Cancer 2003+)。**Circulating Tumor Cells (CTCs)**: <1 in 10^9 cells、liquid biopsy 検出可能 (Phase 421)。**Metastasis genes**: TWIST1, SNAIL (EMT)、MMP9 (invasion)、STAT3。**ITU view**: Metastasis = ρ_tumor の modular flow under tissue boundaries。**Mechanism**: K_cancer^primary tissue → K_cancer^secondary tissue via reduced density transformations、circulating phase は coherent state。
