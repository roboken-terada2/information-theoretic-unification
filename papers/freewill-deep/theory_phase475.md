## 4 つの中心仮説 (H_F1-H_F4)

### H_F1: Decision is operator-algebraic state

```
ρ_decision over action space A:
  H_decision = L²(Actions × Pre-decision_states)
  
Pure state |a⟩⟨a|: deterministic choice
Mixed state Σ p_a |a⟩⟨a|: probabilistic choice (free will?)
```

### H_F2: K_freewill quantifies decision uncertainty

```
K_freewill = -log ρ_decision
⟨K_freewill⟩ = -Σ p_a log p_a = Shannon entropy of action distribution

Determinism: K_freewill = 0
Random: K_freewill = log|A| (max entropy)
Genuine choice: 0 < K_freewill < log|A|
```

### H_F3: Libet readiness potential ⇔ K_freewill pre-decision modular flow

```
Libet 1983: RP -500ms before conscious 'decision'
Schurger 2012: RP = stochastic accumulator noise, not deterministic precursor

ITU view:
  Pre-decision state: ρ_pre = high entropy (many possibilities)
  Pre-decision RP buildup: ρ_pre evolves via modular flow
  At decision: ρ_decision collapses to specific action
  Conscious awareness: lags actual decision (Libet finding)
```

### H_F4: Compatibilism = K_freewill emerges in macroscopic limit

```
Microscopic: deterministic physics (classical) or quantum unitary
Macroscopic: K_freewill > 0 due to:
  (a) Coarse-graining over hidden variables
  (b) Quantum decoherence (Zurek 1991)
  (c) Sensitivity to initial conditions (chaos)
  
Sapolsky 2023 Determined: argues no free will at any scale
ITU intermediate: K_freewill > 0 in macroscopic, but it's emergent
```
