# Phase 477: K_freewill = -log ρ_decision 厳密定義

**ρ_decision over action space**: A = {move left, move right, do nothing, ...}、|A| = action count。**K_freewill = -log ρ_decision = Σ p_a log p_a-related**、⟨K_freewill⟩ Shannon entropy。**Determinism**: ρ_decision = |a*⟩⟨a*| pure state → K = 0。**Random**: ρ_decision = uniform → K = log|A| max。**Genuine choice**: 0 < K < log|A|。**Conditional**: K_freewill | context = entropy given environmental cues。
