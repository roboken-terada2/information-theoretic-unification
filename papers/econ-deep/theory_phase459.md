## 4 つの中心仮説 (H_E1-H_E4)

### H_E1: Market is operator-algebraic state

```
ρ_market = density over (prices × volumes × expectations × behavioral biases × policy)
H_market = L²(Assets × Agents × Information × Time)
```

### H_E2: K_econ measures market efficiency + volatility

```
Efficient market (EMH Fama 1970): ρ_market = uniform over allowed states
  → ⟨K_econ⟩ maximum
Bubble: ρ_market concentrated in narrow speculative subspace
  → ⟨K_econ⟩ low
Crash: ρ_market sudden re-distribution
  → δS = δ⟨K_econ⟩ jump
```

### H_E3: Behavioral biases = K_econ subspace bias

```
Kahneman-Tversky 1979 Prospect Theory:
  Loss aversion (λ ≈ 2.25), reference dependence
  → ρ_market not uniform but biased toward loss-averse subspace
  → ⟨K_econ⟩ < EMH prediction

Thaler 2017 Nobel: nudge architecture
  → small policy → ρ_market subspace shift
```

### H_E4: Policy = K_econ modular flow intervention

```
Fed funds rate change: σ_t^{Fed}(ρ_market) = price/volatility shift
QE/QT: balance sheet expansion → liquidity injection
Forward guidance: expectation modulation → ρ_market subspace pre-emption

ITU view: monetary policy = direct modular flow on ρ_market
Fiscal policy = ρ_government interaction with ρ_market
```
