# Phase 433: Telomere biology (Blackburn-Greider-Szostak Nobel 2009)

**Elizabeth Blackburn + Carol Greider + Jack Szostak Nobel 2009 Physiology**: telomere structure + telomerase discovery。**Telomeres**: TTAGGG repeats at chromosome ends、shorten with each cell division。**Hayflick limit explained**: telomere attrition → senescence trigger。**Telomerase (TERT + TERC)**: lengthens telomeres、active in stem/germ cells + 90% cancers。**Telomere-associated aging diseases**: dyskeratosis congenita、idiopathic pulmonary fibrosis、aplastic anemia。**Telomerase activation therapy**: TA-65 (Astragalus extract) — modest effect、Geron telomerase inhibitor imetelstat (Rytelo FDA 2024.6 for myelodysplastic syndromes)。**ITU view**: telomere length = ρ_chromosome subspace coherence、shortening → K_aging genomic contribution increases。
