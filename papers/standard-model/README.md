### 3.4 Phase 138 — QCD + Confinement + Chiral SSB + Strong CP

Cornell potential V(r) = -α/r + σr with σ ≈ 1 GeV/fm. **Proton mass 938.27 MeV: 0.96% bare quarks + 99.04% QCD chiral SSB** ★ (99% of ordinary matter mass from QCD).
