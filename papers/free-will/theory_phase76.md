## 0. Phase 76 の目的

Phase 75 で **自由意志 = K_self の constraint 能力** と定義しました。Phase 76 では:

1. **前頭前野 (PFC)** の executive control = K_self の生物学的実装
2. **Default Mode Network (DMN)** = 自己生成思考の神経基盤
3. **Soon et al. (2008)**: fMRI で **10 秒前**に決定予測可能
4. **Robert Sapolsky "Determined" (2023)**: hard determinism の現代版
5. **K_self 変動**: ストレス、薬物、睡眠不足、脳損傷の影響
6. Phase 77-78 (倫理・AI) への基盤

中心テーゼ:

> **K_self は脳の特定回路に局在化**: PFC + DMN + ACC + 島皮質。
> これらの統合的活動が「**自由意志度**」 を決定。
> 損傷・薬物・stress で **K_self 度が動的に変動** ⇒ 法的責任と直結。

## 1. 前頭前野 (PFC) ― K_self の中枢

### 1.1 主要領域

| 領域 | 機能 | K-component 対応 |
|---|---|---|
| **DLPFC** (背外側) | 作業記憶, 抑制制御 | K_executive |
| **mPFC** (内側) | 自己参照, 心の理論 | K_self_model |
| **OFC** (眼窩) | 価値評価, 抑制 | K_value |
| **ACC** (前帯状回) | 葛藤監視, 注意 | K_conflict_monitor |
| **vlPFC** (腹外側) | 言語抑制, 反応抑制 | K_inhibition |

### 1.2 Phineas Gage (1848) ― 古典的症例

鉄棒事故で左 PFC を損傷:
- 知能・記憶は保たれた
- 性格が激変: 衝動的, 社会的判断不能
- ⇒ **PFC = 「人格」「意志」 の物理的座席**を示した最初の症例

ITU 解釈: PFC 損傷 ⇒ K_self の物理的破綻 ⇒ free will degree 低下。

### 1.3 PFC 発達 ― 自由意志の年齢依存性

- 出生時: 未発達
- 5 歳: 基本的執行機能
- **10 歳**: 計画立案開始
- **15-18 歳**: 衝動制御発達中 (青年期問題行動)
- **25 歳**: PFC 完成
- 70+ 歳: 部分的衰退

⇒ ITU 予測: free will degree が**年齢で系統的に変動** (児童 0.2 → 成人 0.4 → 高齢 0.3)。

## 2. Default Mode Network (DMN) ― 内的自己

### 2.1 構成

Raichle (2001) の発見:
- **mPFC + PCC + Angular gyrus + Precuneus**
- 「何もしていない時」 にアクティブ
- 自己生成思考、回想、将来計画、心の理論

### 2.2 ITU 解釈

DMN = **K_self の dynamic 維持機構**:
- 外界注意が低下 ⇒ DMN 優位 ⇒ K_self が「自分」 を再構成
- 瞑想・psilocybin で DMN 抑制 ⇒ 自己境界消失感

### 2.3 病的状態と DMN

| 状態 | DMN 異常 |
|---|---|
| うつ病 (Tier 1 #7) | 過活性 ⇒ rumination |
| 統合失調症 | 解離 ⇒ self-other 境界消失 |
| Alzheimer's | 委縮 ⇒ 自己喪失 |
| ASD | 機能低下 ⇒ 心の理論困難 |
| 麻酔下 | 機能停止 ⇒ 無意識 |

## 3. Soon et al. (2008) ― fMRI で 10 秒前予測

### 3.1 実験設定

被験者が任意のタイミングで左右ボタンを押す。fMRI で:
- 内側 PFC + 楔前部 → 60% 精度
- 前帯状回 → 60% 精度
- **両者の組合せで決定 7-10 秒前から予測可能**

### 3.2 結果

| Region | Prediction window |
|---|---|
| Internal mPFC | -7 to -10 s |
| Posterior cingulate | -5 to -7 s |
| Combined | **-10 s** |

⇒ Libet (350 ms) より遥かに長い予測時間 ⇒ **「自由意志はもっと弱い」** という強い実証。

### 3.3 ITU 解釈

- 「意識的選択」 と認識される瞬間は K_self の **後付け説明**
- 真の選択プロセスは PFC + DMN で**秒単位前**に進行
- ただし: K_self は最終 veto を持つ ⇒ free won't は temporal 余裕あり

## 4. Robert Sapolsky "Determined" (2023)

### 4.1 主張

Stanford 神経科学者:
- 「自由意志は完全に幻想」 (hard determinism)
- 全ての行動は遺伝 + 環境 + 過去経験で決定
- 道徳的責任は**物理的に存在しない**
- 法的処罰は「治療」 に変えるべき

### 4.2 ITU 視点での評価

Sapolsky は正しい部分:
- K_self は完全自由ではない (degree < 1)
- 神経科学的予測力は実証されている

Sapolsky が見落とした部分:
- **K_self degree は 0 ではない** (Libet veto, PFC 介入)
- 「責任なし」 では社会機能が崩壊 ⇒ 実用的に compatibilism が必要
- 量子的 indeterminacy の余地

ITU 視点: **「弱い free will」 = compatibilism** が中道。

## 5. K_self degree の変動要因

### 5.1 一時的変動

| 条件 | free will degree 変化 |
|---|---|
| 睡眠不足 (24h+) | -30 to -50% |
| アルコール (BAC 0.08) | -40% |
| 強いストレス | -20 to -40% |
| マインドフルネス瞑想 | +10 to +20% |
| 朝の高 cortisol | +15% (時間帯依存) |

### 5.2 慢性的変動

| 条件 | 影響 |
|---|---|
| 薬物依存 | K_self の craving 制御不能 → degree -50% |
| ADHD | K_attention 不全 → degree -25% |
| 児童期トラウマ | K_self の構造的弱体化 |
| 慢性うつ病 | K_reward 不全 → 意思決定能力↓ |
| 慢性精神安定剤 | 数値曖昧 (混合的影響) |

### 5.3 脳損傷別

| 損傷部位 | free will 影響 |
|---|---|
| **PFC 全体** | 最も深刻 (Gage 型) |
| OFC | 衝動制御不能 |
| Amygdala | 恐怖反応欠失 (Urbach-Wiethe) |
| DMN 委縮 (Alzheimer's) | 段階的喪失 |
| Hemi-neglect (parietal) | 半側自己消失 |
| Split-brain | 二人格化 (Gazzaniga) |

⇒ **K_self degree は静的でなく、生物学的・心理的状態で連続変動**。

## 6. 法的責任との接続 (Phase 77 への伏線)

### 6.1 現代刑法の前提

「**犯人は理性的選択をして犯罪を行った**」 という free will 仮定。

### 6.2 Insanity Defense (心神耗弱)

- M'Naghten Rules (1843): 自分の行動を理解できなかった場合
- Model Penal Code: substantial capacity 喪失
- ⇒ K_self degree が**著しく低下**の認識

### 6.3 ITU 視点の改革提案

- **K_self degree の動的測定** (fMRI, EEG biomarker) ⇒ 司法判断
- ストレス・薬物・脳損傷の物理的客観評価
- 「責任 vs 治療」 のグラデーション化

⇒ Phase 77 で詳述。

## 8. Phase 76 の結論

1. **K_self は脳の局在回路**: PFC + DMN + ACC + 島皮質
2. **Soon 2008**: fMRI で **10 秒前**に決定予測 ⇒ 強い決定論支持

5. **法的責任**: ITU 視点での改革余地大 (Phase 77)
