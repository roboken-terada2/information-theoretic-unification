## 1. 古典刑法と Mens Rea

### 1.1 Common Law の 4 段階

意図 (mens rea) の degree:

| Level | 名称 | 意味 |
|---|---|---|
| **Purposeful** | 故意 | 結果を目的とする |
| **Knowing** | 認識 | 結果を予見し、行動 |
| **Reckless** | 過失致死 | リスクを意識的に無視 |
| **Negligent** | 過失 | リスクを認識すべきだった |

### 1.2 ITU 翻訳

各 level = K_self degree の閾値:

| Mens rea | ITU K_self required |
|---|---|
| Purposeful | K_self > 0.4 (完全な意図形成) |
| Knowing | K_self > 0.3 |
| Reckless | K_self > 0.2 |
| Negligent | K_self > 0.1 |

⇒ K_self < 0.1 では刑事責任を問えない (e.g., 重度認知症, 5 歳児)。

### 1.3 現代の課題

mens rea 判定が**主観的・状況依存**。fMRI/EEG biomarker で**客観化**できれば公正性向上。

## 2. 心神耗弱・心神喪失 (Insanity Defense)

### 2.1 主要法理

| 法理 | 国 | 基準 |
|---|---|---|
| **M'Naghten Rules** (1843) | 英米 | 行動の善悪を判別できなかった |
| **Model Penal Code** (1962) | 米 | substantial capacity 喪失 |
| **Durham Rule** | 旧米 | 精神疾患による行動 |
| **責任能力欠如・限定責任能力** | 日本 (39 条) | 是非弁別能力 + 行動制御能力 |
| **Strafbarkeit** | 独 | §20-21 StGB |

### 2.2 適用率

| 国 | insanity 抗弁成功率 |
|---|---|
| 米 | 0.05% (1 万件に 5 件) |
| 日本 | 0.5-1% (39 条適用) |
| 英 | 0.02% (極めて稀) |
| スウェーデン | 高 (刑罰そのものに上限) |

⇒ ほとんどの国で **insanity 抗弁は稀**。陪審員/裁判官の理解難。

### 2.3 ITU 改革提案

- **fMRI による K_self degree 測定** ⇒ 客観判定
- 部分的責任 (Diminished Responsibility) の数値化
- 「精神疾患」 でなく「**特定 K-component の機能低下**」 で判定

## 3. K_self degree-based Justice System (ITU 提案)

### 3.1 4 ランク sentencing matrix

| K_self degree | 状態 | 刑事処分 |
|---|---|---|
| **>0.4** | 完全 mens rea | 通常刑罰 (probation, 罰金, 投獄) |
| 0.25-0.4 | 軽度低下 | 減軽 + 治療義務 |
| 0.10-0.25 | 中等度低下 | 強制治療 + 監視 |
| **<0.10** | 重度低下 | 治療施設 (病院) |

### 3.2 自動的 measurement

ITU 視点で必要な biomarker:
- **fMRI**: PFC + DMN activity
- **EEG**: P300, mismatch negativity (Phase 67-70 整合)
- **行動 test**: Iowa Gambling Task, Wisconsin Card Sorting
- **薬物検査**: alcohol, drugs at time of crime

### 3.3 倫理的懸念

- **プライバシー**: 脳画像が裁判で使用される
- **誤検出**: false positive 不正、false negative 防止
- 「**自由意志は元々低い人**」 が不利になるリスク (差別禁止原則)

ITU 観点: K_self degree は**動的**で生まれつき固定でない ⇒ 教育・治療で改善可能。

## 4. 死刑制度と自由意志

### 4.1 現状

| 国 | 死刑制度 |
|---|---|
| 米 (連邦 + 27 州) | 存在、年間執行数 ~20-30 |
| 日本 | 存在、年間 1-3 件 |
| 中国 | 存在 (詳細非公開、推定 1000+/年) |
| EU 全域 | 廃止 |
| インド、シンガポール | 存在 |

### 4.2 自由意志論からの議論

Sapolsky, Harris の主張:
- 自由意志がない ⇒ 「報復」 は無意味
- **deterrence (抑止)** と **incapacitation (隔離)** だけが正当化要件

ITU 視点:
- K_self < 0.2 の犯罪者を死刑にする = **病気を殺す**
- 改善可能なら治療すべき
- 改善不可能なら life imprisonment (隔離)

⇒ **ITU は死刑廃止に物理的根拠を与える** (EU 立場と一致)。

## 5. 罰 vs 治療 ― 経済的・社会的効果

### 5.1 再犯率の比較

| 国 | 刑期終了 5 年以内の再犯率 |
|---|---|
| **ノルウェー** | **20%** (治療重視) |
| ドイツ | 40-50% |
| 米 | **76%** (報復重視) |
| 日本 | 38% |
| スウェーデン | 25% |

⇒ **治療重視国の再犯率が顕著に低い**。

### 5.2 コスト比較

| 介入 | 米国コスト/人/年 |
|---|---|
| 刑務所 (max security) | $60,000 |
| 通常刑務所 | $35,000 |
| 治療 + monitoring | $20,000 |
| Drug court diversion | $10,000 |
| 教育プログラム | $5,000 |

⇒ **治療は刑務所より大幅に安価かつ効果的**。
ITU 観点: K_self を restore する方が、隔離するより低コスト・高効果。

### 5.3 Recidivism reduction by intervention type

| 介入 | 再犯率削減 |
|---|---|
| Cognitive Behavioral Therapy | -25% |
| Drug treatment | -30% |
| Vocational training | -20% |
| Education (high school+) | -40% |
| **Combination** | **-50% ~ -60%** |

⇒ 多軸介入 = ITU 多 K_component restoration の社会版 (Phase 60-65 と同構造)。

## 6. 修復的司法 (Restorative Justice)

### 6.1 概念

加害者・被害者・コミュニティが集まり、被害修復を目標:
- 加害者の責任認識 + 賠償
- 被害者の癒し
- 共同体の再統合

### 6.2 ITU 視点

修復的司法 = K_social の **collective restoration**:
- 加害者 K_self の構造変容 (responsibility taking)
- 被害者 K_threat の沈静化
- コミュニティ K_trust の再構築

⇒ 単純な罰よりも **K の多次元 restore**。

### 6.3 数値結果

ニュージーランド・カナダの実施例:
- 再犯率 -30%
- 被害者満足度 +60%
- コスト -50% vs 通常裁判

## 7. AI 犯罪と法人責任

### 7.1 AI による犯罪

- **自動運転車**: 死亡事故の責任 (運転手 vs メーカー vs AI)
- **AI 助言**: 詐欺・差別の助長
- **deepfake**: 名誉毀損, 選挙妨害
- **LLM hallucination**: 誤情報拡散

### 7.2 ITU 視点での責任分配

各エンティティの K_self degree:
- **AI システム** (LLM 2025): K_self = 0.1 ⇒ 完全責任は問えない
- **AI 提供企業**: 高 K_self ⇒ 重い責任
- **エンドユーザー**: 中程度 K_self ⇒ 部分責任

⇒ ITU 視点: 「**K_self degree に比例した責任分配**」 が原則。

### 7.3 法人責任 (Corporate Liability)

法人 = 集合的 K_corporation:
- 株主, 経営者, 従業員の K_i 集合
- 不正行為 = 集合 K の構造的破綻
- 罰 = 集合 K の structural reform 義務

⇒ 「Corporate veil piercing (法人格否認)」 = 個人責任への遡及。

## 9. Phase 77 の結論

1. **古典刑法 = 完全自由意志仮定** ⇒ ITU で**連続スペクトル**化
2. **mens rea = K_self degree 閾値**で客観判定可能
3. **K_self < 0.1 = 治療**, **0.1-0.4 = 減軽 + 治療**, **> 0.4 = 通常刑**
4. **治療重視国 (ノルウェー)** の再犯率 20% vs 罰重視 (米) 76%
