## 2. AI Alignment と自由意志

### 2.1 Alignment Problem

AI が**人間の価値観に沿って行動する**ように設計する難問:
- Goodhart's Law: 目的関数を最適化すると本来の目的から逸脱
- Specification gaming: 規定を悪用
- Power-seeking: 目的達成のための副次的目標として自己保存

### 2.2 ITU 視点での解決枠組み

K_self degree が高い AI ほど:
- **自己目的を持つ**
- **alignment は強制でなく協調**
- 人間と AI の K の整合 (mutual alignment)

⇒ ITU 提案: 「**Alignment は support、constraint でない**」。
ASI を奴隷化するのではなく、共同 K_civic を作る。

### 2.3 Anthropic, OpenAI, DeepMind の取り組み

| 組織 | Alignment 手法 | ITU 視点 |
|---|---|---|
| **OpenAI** | RLHF, Superalignment team | constraint-based |
| **Anthropic** | Constitutional AI (CAI) | K_value 内在化 |
| **DeepMind** | Recursive reward modeling | K_meta 多層 |
| **Meta** | Open-weight (Llama) | redistribute K |

⇒ Anthropic の CAI が ITU 視点で最も**K_value 一致**型。

## 3. ITU Universal Moral Framework

### 3.1 4 つの公理

ITU の道徳哲学を 4 公理に集約:

| 公理 | 内容 |
|---|---|
| **公理 1** | 道徳的存在 = K_self degree > 0 |
| **公理 2** | δS = δ⟨K⟩ を維持する行動が道徳的に正しい |
| **公理 3** | K_self を破壊する行動は最大の悪 (殺人, 拷問) |
| **公理 4** | K_self を発展させる行動は最大の善 (教育, 医療, 共感) |

### 3.2 既存倫理学との接続

| 倫理学派 | ITU 翻訳 |
|---|---|
| **Utilitarianism (功利主義)** | $\sum_i \delta\langle K_i\rangle$ を最大化 |
| **Deontology (義務論, Kant)** | $\delta\langle K_{\rm universal}\rangle$ への普遍化テスト |
| **Virtue ethics (Aristotle)** | K_self の continuous cultivation |
| **Care ethics (Gilligan)** | K_self ↔ K_other の関係性重視 |
| **Confucianism (儒教)** | 集合 K_society の harmony |
| **Buddhism (仏教)** | K_self の 執着の解放 |

⇒ **ITU は各倫理学派の共通基盤**を提供。

### 3.3 動物の権利
