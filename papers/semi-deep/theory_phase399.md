# Phase 399: TSMC 2nm + GAA transistors

**TSMC N2 (2nm) node** が 2025 mass production 開始予定。**GAA (Gate-All-Around)** transistors への architecture 移行。

## TSMC N2 specs (Apple A19 2025 predicted)

```
Technology: GAA nanosheet
Density: ~330M transistors/mm² (estimate)
Power: -25-30% vs N3
Performance: +10-15% vs N3
EUV: yes (multi-patterning + high-NA potential)

First products:
  Apple A19 Pro (iPhone 17 Pro 2025.9)
  Qualcomm Snapdragon X Elite Gen 2
  AMD CPUs (2026+)
```

## GAA vs FinFET

```
FinFET (TSMC N7-N3, Samsung 14LPP-3GAE):
  Vertical fin with gate on 3 sides
  Used 2011-2024

GAA Nanosheet (TSMC N2+):
  Multiple nanosheets stacked
  Gate completely surrounds channel
  Better electrostatic control
  Lower leakage at small dimensions
```

## Samsung 3nm GAA (2022)

```
Samsung 3GAE (June 2022): world's first GAA
But lower yields than TSMC N3
2024: 3nm GAP refresh

ITU view: GAA = K_semi gate dimension expansion
  3D-surrounding gate → effective gate area increases
  K_semi contribution from gate increases
