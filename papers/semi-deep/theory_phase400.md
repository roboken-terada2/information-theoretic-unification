# Phase 400: ASML EUV High-NA (0.55 NA)

**ASML High-NA EUV (TWINSCAN EXE:5000)** が次世代 lithography の象徴。

## High-NA EUV specs

```
NA (numerical aperture): 0.55 (vs standard 0.33)
Wavelength: 13.5 nm (same)
Resolution: ~8 nm (vs 13 nm standard)

Anamorphic optics: 8x reduction in scan direction, 4x in cross
→ field size: 26 × 16.5 mm (vs 26 × 33 mm standard)
→ throughput: 220 wafers/hour
```

## Pricing & shipment

```
Single machine: $400M
2024.1: First shipped to Intel
2024-25: TSMC, Samsung 受領
Annual production: ~50 systems

Total addressable market: $20B/year+
ASML monopoly (>80% EUV市場)
```

## Impact on Moore's Law

```
High-NA enables:
  Single exposure for 14-16 nm pitch
  vs multi-patterning for sub-15 nm

→ Lower cost per layer
→ Faster cycle time
→ Process node continuation
```

## ITU view

```
High-NA = K_semi resolution enhancement:
  ⟨K_semi⟩ ≈ log(wafer / R²)
  R: 13 → 8 nm
  K_semi gain: log(13²/8²) ≈ 1 nat ≈ 1.4 bits per device area
```
