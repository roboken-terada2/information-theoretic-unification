# Phase 406: Neuromorphic computing (Intel Loihi 2, IBM TrueNorth)

**Neuromorphic chips** = brain-inspired architecture。Post-Von-Neumann。

## Intel Loihi 2 (2021)

```
1 million neurons, 120 million synapses
Process: Intel 4 (7nm)
Asynchronous, event-driven (spike-based)
Power: 100x less than equivalent GPU for SNN

Used in: Mihaylov-2024 olfactory chip, Sandia Pohoiki Springs (768 Loihi chips)
```

## IBM TrueNorth (2014, predecessor)

```
1 million neurons, 256 million synapses
65 mW total power
Architecture inspired by mammalian cortex

Production stopped 2017, research only
```

## SpiNNaker 2 (Manchester, 2024)

```
Massively parallel ARM cores
Used in Human Brain Project
EU funded ~€1B (2013-2023)
```

## ITU view: K_neuromorphic

```
Brain has K_self ~10^10 bits (Tier 1+ #2):
  Spike-based, event-driven, sparse

Neuromorphic chip targets:
  ρ_chip ≈ ρ_brain for specific tasks
  K_neuromorphic ≈ K_brain (subset)

Energy efficiency: K_per_joule >> K_GPU_per_joule
