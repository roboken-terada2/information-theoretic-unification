# Phase 367: Cogitate Consortium 2019-23 history + Nature 2023

**Cogitate Consortium** は Templeton World Charity Foundation 資金 ($20M、2019-23)
の意識理論 adversarial collaboration。Lucia Melloni (Max Planck Frankfurt) +
Christof Koch (Allen Institute) + Giulio Tononi (Wisconsin) リード。

## Cogitate 経緯

```
2018: Templeton $20M grant 承認
2019: Consortium 結成、design phase (Melloni, Koch, Tononi)
  IIT (Tononi) と GNW (Dehaene) の双方の予測を **pre-registered** 化
2020-22: データ収集 (6 labs、256 participants、3 modalities)
  - MEG (magnetoencephalography)
  - intracranial EEG (epileptics surgery 時)
  - fMRI (7T)
2023.4-5: 解析完了
2023.6: Nature 発表 (Cogitate Consortium 著者)
2023.9.16: 124 神経科学者 letter で IIT "pseudoscience" 批判
2024-25: Follow-up analyses 継続
```

## Cogitate 2023 Nature 主要結果

```
Pre-registered predictions:
  IIT: posterior γ activity sustained throughout stimulus
  GNW: prefrontal ignition at late onset (250-500ms)

Observed (256 participants):
  Visual stimuli (faces, objects) presented for various durations
  Reportable (conscious) vs Unreportable (subliminal) compared

Results (simplified):
  ✓ IIT prediction posterior γ: partially supported but weaker than expected
  ✓ GNW prediction prefrontal ignition: detected but timing differs
  ✗ Neither theory completely explains observed data
  ⇒ Both theories partially falsified
```

## 主要批判 (2023.9 letter)

```
124 神経科学者が Nature に open letter:
  "IIT is testable but currently unfalsified ≠ proven"
  "Pseudoscience" は強すぎる言葉だが、注意喚起意義あり

Tononi (IIT 提唱者) からの反論:
  "Adversarial collaboration design 自体が不公平"
  "IIT prediction を fully test していない"

Christof Koch (Allen Institute, IIT 推奨者) からの応答:
  "Cogitate results は IIT を refine する機会"
```

## ITU の position

```
ITU は Cogitate adversarial test に **第 3 候補** として参戦提案:

  K_self framework は:
  - IIT Φ より broader (global self-model)
  - GNW workspace より narrower (focused on self vs other)
  - 中間階層で両者統合

Cogitate Round 2 (Phase 368 で詳述) で:
  IIT vs GNW vs ITU 3-way pre-registered test
  各理論の K_self/Φ/workspace 別個に測定
  予測の重なり/相違を統計的に分離
```

## Templeton funding rationale

```
John Templeton Foundation (1996-, $3.5B endowment):
  "Big questions" を adversarial collaboration で研究
  Religion + Science 交差テーマも fund

Cogitate は flagship project (最大規模):
  $20M / 6 labs / 5 yrs / 3 modalities / 256 participants
  Open Science 完全 commitment (OSF 全データ public)
```

ITU framework が Templeton 第 2 round funding を獲得できれば、
Cogitate Round 2 with 3-way adversarial が現実化。
