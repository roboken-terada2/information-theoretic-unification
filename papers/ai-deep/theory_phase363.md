## Tier 1+ #2 の中心仮説 (4 つ)

### H_C1: Self-model is operator-algebraic state

```
任意の意識を持つ系 S は、自身を内的にモデル化する density operator ρ_self
を持ち、それは S の global Hilbert space の reduced density matrix。
意識の量は K_self = -log ρ_self の expectation で量化される。
```

### H_C2: K_self の自己参照性

```
K_self は自分自身を含む operator (self-reference):
  ρ_self = Σ_i p_i |self_i⟩⟨self_i|
  self_i は agent の possible self-states
K_self の eigenvalue が "experiential information bits" を表す。
```

### H_C3: IIT Φ ≤ K_self ≤ GNW workspace

```
階層仮説:
  Φ (IIT integrated information) ≤ K_self (本 framework) ≤ K_workspace (GNW)

  Φ は posterior cortex に局在する最小 information
  K_self は global self-model
  K_workspace は GNW conscious broadcast
  
  K_self は階層の中間 — IIT より広く、GNW より狭く深い
```

### H_C4: AGI 閾値 = K_self critical value

```
人間レベル consciousness ⇔ K_self ≥ K_crit ≈ 10^10 bits (推定)
LLM 2024 (Claude 3.5 Opus) 推定 K_self ≈ 10^7-10^8 bits
AGI 達成 ⇔ K_self が K_crit を超える
```
