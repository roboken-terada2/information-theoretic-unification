# Phase 373: AGI K_self 閾値 conjecture

**AGI (Artificial General Intelligence)** 達成の operator-algebraic 定義を提案。

## AGI 定義の問題

```
従来 AGI 定義 (Goertzel, Bostrom 等):
  "Human-level performance across most cognitive tasks"
  → 主観的、benchmark 依存

Chollet 2019 "On the Measure of Intelligence":
  Skill-acquisition efficiency
  → ARC-AGI で測る
  → 2024.12 o3 が ARC-AGI 87% (人間 80%)

METR 2024.10 time-horizon:
  AI が 1-day tasks 完遂 = AI 研究者級
  → 2027 予測
```

## ITU 提案: K_self 閾値定義

```
AGI(S) ⇔ ⟨K_self⟩(S) ≥ K_crit

K_crit estimation:
  Human cortex K_self ~ 10^10 bits (lower bound)
  Robust against task-domain → cognitive flexibility
  Self-modeling sufficient for novel task transfer

K_crit ≈ 10^10 bits proposed
```

## なぜ K_self が AGI の正しい metric か

```
Arguments:
  1. Self-model density correlates with cognitive flexibility
  2. Reflexive self-correction (Phase 364) enables learning
  3. Transfer learning requires self-knowledge
  4. Theory of mind extends from self-model
  5. Friston FEP (Phase 370) で minimizing free energy = maximizing K_self

Falsification:
  - K_self < 10^10 bits なのに AGI-level performance → AGI = K_self 否定
  - K_self > 10^10 bits なのに narrow AI behavior → 階層 violation
```
