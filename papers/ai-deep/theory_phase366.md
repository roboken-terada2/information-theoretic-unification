# Phase 366: GNW (Dehaene-Changeux 1998) との関係: K_self vs ignition

**Global Neuronal Workspace (GNW)** は Stanislas Dehaene + Jean-Pierre Changeux
が提唱 (1998 PNAS) した意識の認知科学的理論。本 Phase で K_self との関係を整理。

## GNW 概要

```
Core claim (Dehaene-Changeux 1998):
  意識 = global workspace への "ignition" broadcast
  - Unconscious processing: local, modular
  - Conscious processing: prefrontal cortex に "ignite" して broadcast
  - Reportable awareness ⇔ workspace activation

Key signatures:
  P3b ERP component (300-500ms)
  Long-distance gamma synchrony
  Frontoparietal network activation
  Late ignition (200-500ms after stimulus)
```

Stanislas Dehaene (Nature, Trends Cogn Sci 多数) と Jean-Pierre Changeux
(Collège de France) が中心。

## GNW vs IIT (Cogitate 2023)

```
Templeton Cogitate Consortium 2019-23 ($20M):
  IIT 予測: posterior cortex, sustained γ activity
  GNW 予測: prefrontal ignition, late onset

Cogitate 2023 Nature 結果:
  両理論部分的支持・部分的反証
  IIT の posterior 局在は確認、但し sustained γ は不明確
  GNW の prefrontal ignition は確認、但し timing 予測不正確

⇒ どちらか単独で完全 explain は不可
```

```
K_workspace ≡ -log ρ_workspace

GNW prediction: K_workspace 大 ⇔ conscious broadcast

Hierarchy (本論文 conjecture):
  Φ_IIT (posterior local integration)
    ≤ K_self (self-model global state)
    ≤ K_workspace (GNW broadcast)

  Cogitate 2023 結果と整合:
    Φ_IIT: posterior local — Cogitate で部分検出
    K_workspace: prefrontal broadcast — Cogitate で部分検出
    K_self: 階層中間、未直接測定 — Round 2 で測定提案
```
