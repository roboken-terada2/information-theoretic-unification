## 0. Phase 104 の目的

Phase 103 で都市 = K-OS と定式化。Phase 104 では:

1. **AGI 都市**: 2035-2045 AGI による都市運営
2. **持続可能性指標**: ISO 37120, IPCC, SDG 11
3. **15 分都市** (Chrono-urbanism)
4. **全 Tier 1 #1-#15 のスマートシティ統合表**
5. **デジタルツイン都市運営**

中心テーゼ:

> スマートシティ = ITU polytope **全 vertex の応用ハブ**。
> AGI 都市 (2040+) は **K_city = K_self^{city-scale}**: 都市自身が学習・適応する K-agent。
> 持続可能性 = K_environmental ⊗ K_social ⊗ K_economic の同時最適化。

## 1. AGI 都市シナリオ

### 1.1 5 段階の AI 都市

| 段階 | 期間 | 機能 |
|---|---|---|
| L1 デジタル化 | 2010-2020 | スマートメーター、IoT、e-government |
| L2 統合 | 2020-2030 | 都市 OS、デジタルツイン、MaaS |
| **L3 予測** | **2030-2040** | **AI 予測保全、需要予測、災害予測** |
| **L4 自律** | **2040-2050** | **AGI 都市運営 (大半の決定 AI)** |
| **L5 ASI 都市** | **2050+** | **超人類的最適化、市民は consultive role** |

### 1.2 ITU 視点: K_city の自律性 degree

$$
K_{\text{city, autonomy}}(t) = \frac{\text{AI-decided}}{\text{total decisions}}
$$

| 年 | 自律 degree |
|---|---|
| 2024 | 0.05 (交通信号等) |
| 2030 | 0.15 (予測保全 + 緊急応答) |
| 2035 | 0.30 |
| **2040** | **0.50 (AGI 統制)** ★ |
| 2050 | 0.75 |
| 2060 (ASI 都市) | **0.90+** |

⇒ **2040 で半数以上の都市決定が AI** ― 公共政策最大の革命。

### 1.3 AGI 都市の機能

- 交通: リアルタイム最適化、自動運転統合 (Tier 1 #13 連動)
- エネルギー: DERMS + 需要予測 (#10, #15)
- 健康: 予防医療、流行検知 (#5-#7)
- 治安: 異常検知、予防介入 (#3)
- 緊急: 災害予測 + 自動避難 (#11, #15)
- 教育: 個別化学習プラン
- 経済: 局所市場最適化 (#8)

## 2. 15 分都市 (Chrono-urbanism)

### 2.1 概念 (Carlos Moreno 2016)

すべての生活機能 (仕事、買い物、医療、教育、娯楽) が **徒歩 / 自転車 15 分以内**:
- パリ Mayor Hidalgo 旗艦政策 (2020-)
- メルボルン、Ottawa 採用
- バルセロナ Superblocks

### 2.2 ITU 視点

$$
K_{\text{accessibility}} = \frac{\text{services within 15min}}{\text{total services needed}}
$$

→ **K_resident の生活 K-flow を 15 min radius 内に圧縮**。

### 2.3 効果 (パリ 2020-2024)

- 自動車交通量 −40%
- 自転車利用 +50%
- 大気質改善 +30%
- 市民満足度 +25%

### 2.4 課題

- ジェントリフィケーション
- 自動車ロビー反発
- 全都市で一様適用は困難

## 4. 持続可能性指標

### 4.1 ISO 37120 + 37122 (Smart Cities Standards)

世界基準:
- **ISO 37120** (2014, 2018 更新): 100+ indicators
- **ISO 37122** (2019): スマートシティ特化 80 indicators
- **ISO 37123** (2019): レジリエンス都市

### 4.2 UN SDG 11 (持続可能な都市)

| SDG 11 ターゲット | 2030 目標 |
|---|---|
| 11.1 安全な住居 | スラム撤廃 |
| 11.2 公共交通 | アクセス全市民 |
| 11.3 包摂的都市化 | 計画市民参加 |
| 11.4 文化遺産 | 保護 |
| 11.5 災害死者 | 大幅削減 |
| **11.6 大気質** | **PM2.5 削減** |
| 11.7 公共空間 | 緑地確保 |

### 4.3 主要持続可能都市指標

| 指標 | Singapore | Copenhagen | Tokyo | New York | 雄安新区 |

### 5.2 ITU 視点

デジタルツイン都市 = **K-state 同型写像**:

$$
K_{\text{physical city}}(t) \xleftrightarrow{\phi} K_{\text{virtual city}}(t)
$$

→ シナリオシミュレーション、政策効果予測、緊急時対応。
