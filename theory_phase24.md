# Phase 24: Bullet Cluster N体シミュレーション — ITU の空間質量分布検証

## 1. 動機

Phase 18 と 20 で銀河団質量は MOND + ハイブリッドで再現できることを示した。
しかし**Bullet Cluster** (1E 0657-558) は別の問題を提起:

**観測 (Clowe et al. 2006)**:
- 2 つの銀河団が衝突した直後の系
- **X 線輝度ピーク** (= 高温ガス、バリオン物質の ~90%): 衝突で減速、ガス雲が衝突中心に滞留
- **重力レンズ質量分布** (= 全質量): 衝突せず通過、ガスの**外側**にオフセット

→ 「ガスではない collisionless な暗黒物質が存在する」直接証拠

**MOND/Verlinde の問題**: 質量はバリオン (= ガス) と一致するはず。しかし**観測のオフセット**は MOND では説明困難。

**ITU 仮説**: Phase 22 の「凍結 QECC 情報」が **collisionless DM** として振る舞うなら、観測のオフセットを再現できる。本 Phase で実際の N 体シミュレーションで検証する。

## 2. 物理モデル

### 2.1 4 種の粒子
2 つの銀河団 A, B、それぞれが「ガス」と「DM 様 (凍結 QECC)」を持つ:
- $\{$ガス_A, ガス_B$\}$: 重力 + **流体的摩擦** (衝突時にガス雲同士は相互作用)
- $\{$DM_A, DM_B$\}$: 重力のみ (**collisionless**)

### 2.2 運動方程式

質量 $m_i$, 位置 $\vec r_i$, 速度 $\vec v_i$ の粒子について:
$$\vec a_i = -G \sum_{j \neq i} \frac{m_j (\vec r_i - \vec r_j)}{(|\vec r_i - \vec r_j|^2 + \epsilon^2)^{3/2}} + \vec f_{\rm drag}(i)$$

$\vec f_{\rm drag}$ はガス粒子のみに働き、相手側ガス雲と接近すると速度を相対速度に向けて減衰させる (= ガス流体間の運動量交換)。

### 2.3 初期条件
- 銀河団 A: 中心 $(-r_0, 0)$, 速度 $(+v_0, 0)$
- 銀河団 B: 中心 $(+r_0, 0)$, 速度 $(-v_0, 0)$
- 各銀河団内: Plummer 風球対称分布
- $r_0 \sim$ 1 unit, $v_0 \sim$ 1 unit (無次元化)

## 3. 予言

### 3.1 ITU hybrid (collisionless DM 含む)
- 衝突後: ガスは中央に滞留、DM は通過
- **質量中心 (= DM 中心) はガス中心からオフセット**
- 観測される Bullet Cluster と整合

### 3.2 MOND-only (DM なし)
- ガスのみが質量源
- 質量中心 = ガス中心
- **オフセットなし** → 観測と矛盾

### 3.3 定量予言
- オフセット距離: ~0.5–1 unit (初期半径の半分程度)
- ガス中心の運動: 中央に滞留 (相対速度 → 0)
- DM 中心の運動: ほぼ通過 (相対速度保存)

## 4. ITU 枠組み内での解釈

凍結 QECC が collisionless DM として振る舞うのは:
- Phase 22 で示した「インフレーション期の情報凍結」
- 凍結ビット間相互作用が**重力のみ** (= 量子情報の trace property)
- ガス (= 通常物質) は電磁相互作用で衝突するが、凍結情報は背景時空にしか結合しない

これは「**ITU 流のダークマター = 量子情報状態**」の自然な性質。

## 5. シミュレーション計画

### Part A: 衝突進化の可視化
- $t = 0, t_{\rm mid}, t_{\rm end}$ の 3 スナップショット
- 4 つの粒子集団を色分けで表示

### Part B: 中心位置の時間進化
- ガス_A, ガス_B, DM_A, DM_B の centroid 位置を時系列で
- ガスは衝突で滞留、DM は通過する様子

### Part C: 終状態のオフセット
- 全 DM 中心 vs 全ガス中心の距離
- 観測 Bullet Cluster との比較 (~150 kpc 程度のオフセット)

### Part D: MOND-only 比較
- DM 粒子なしバージョン
- 質量中心 = ガス中心 → オフセット 0
- 観測との矛盾を明示

## 6. 限界

⚠️ 単純化:
- 2D (実際は 3D)
- 簡略化したガス流体 (実際は流体力学 + 衝撃波)
- 重力ソフトニングで近距離発散を回避
- 観測の精密スケール ($\sim$ 0.7 Mpc オフセット) は無次元単位で代用

✅ 検証できる本質:
- collisionless DM があると質量中心がオフセット
- ITU 凍結 QECC は同じ振る舞いをする可能性
- MOND-only ではオフセットが出ない
